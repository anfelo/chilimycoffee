Setup Environment

Before we start coding the actual game we need to make sure that we have all the tools in place, installed in our computer.

We will be making a game in C and for that to happen, our computer must be able to compile the code that we write. Luckily, almost every operating system comes with a C compiler installed that we can use. Depending on which operating system (OS) you are running on, you will use one or the other. But don't worry I will guide you through the entire process no matter what computer you are using.

Let us first check that you are all set.

macOS

On macOS the C compilers are typically installed as part of the Xcode Command Line Tools. You have two options here.

Clang (from the LLVM Project) is the default C, C++, and Objective-C compiler on macOS. It is included with the Xcode Command Line Tools. You can verify that it is installed by running: 

$ clang --version

The other alternative is GCC (GNU Compiler Collection) which can be installed using a package manager like Homebrew like this: $ brew install gcc

Once installed you can verify the version by running on your terminal:

$ gcc --version

If none of those is available in your machine, check the installation guide.

Linux

In Linux it depends on the distribution but the most common one GCC (GNU Compiler Collection). It is the default compiler in most distros. Make sure that it is installed by running the following command: 

$ gcc --version

An alternative would be clang from the LLVM Project.You can also check if it is installed: 

$ clang --version

If none of those are installed in your Linux machine, check the installation guide.
